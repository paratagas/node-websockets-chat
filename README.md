# node-websockets-chat

### Chat created using Javascript, Node.js and Websockets
---

## Actions:

### Setup
```sh
$ npm i
```

### Start
```sh
$ npm start
```

### Test
```sh
$ npm run test
```

### Lint (./ and ./scripts/src)
```sh
$ npm run lint
```

### Lint and fix problems
```sh
$ npm run lint:fix
```

### Create documentation (./docs)
```sh
$ npm run jsdoc
```
